# 🚗 Автошкола: Симулятор экзамена ПДД

Веб-приложение для подготовки к сдаче теоретического экзамена в ГИБДД.

## 📋 Описание

Проект разработан в рамках курсовой работы по дисциплине «Кроссплатформенная среда исполнения программного обеспечения».

### Возможности:
- ✅ 5 экзаменационных билетов по 20 вопросов
- ✅ Реальные вопросы из экзаменов ПДД 2024
- ✅ Максимум 2 ошибки для сдачи
- ✅ Красивый адаптивный интерфейс
- ✅ Полная контейнеризация

## 🛠 Технологии

- **.NET 8.0** — фреймворк
- **ASP.NET Core Blazor Server** — веб-интерфейс
- **Entity Framework Core 8** — ORM (Code-First подход)
- **SQLite** — база данных
- **Docker & Docker Compose** — контейнеризация
- **Git** — система контроля версий
